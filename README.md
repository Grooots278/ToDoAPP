# ToDo Application

Полнофункциональное приложение для управления задачами с использованием:

## Технологический стек

### Backend
- ASP.NET Core Web API
- Entity Framework Core
- PostgreSQL (в Docker)
- Swagger для документации API

### Frontend
- React 18
- TypeScript
- Axios для HTTP запросов

### Инфраструктура
- Docker
- Docker Compose
- PostgreSQL

## Функциональность

- ✅ Создание задач (POST)
- ✅ Просмотр всех задач (GET)
- ✅ Просмотр отдельной задачи (GET by ID)
- ✅ Обновление задач (PUT)
- ✅ Удаление задач (DELETE)
- ✅ Отметка выполнения
- ✅ Редактирование на месте
